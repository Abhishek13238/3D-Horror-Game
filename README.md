# 3D Horror Survival Game

A first-person 3D horror survival game developed in Unity using C#. Explore a dark environment, collect all 6 hidden statues, unlock the exit gate, and escape while avoiding a dangerous enemy that patrols and chases the player.

## Features

* Main Menu System
* First-Person Player Controller
* Collectible Statue Objective System
* UI Counter for Collected Statues
* Enemy AI with Patrol and Chase Behavior
* Gate Unlock Progression System
* Game Over and Game Complete Screens

## Gameplay

The player spawns outside a house in a dark environment and must search for 6 hidden statues. Each collected statue updates the UI counter. After collecting all statues, the exit gate opens with an text message. The player must then reach the gate to complete the game while surviving the enemy chase.

## Controls

* WASD — Move
* Mouse — Look Around

## Built With

* Unity
* C#

This project was created as part of my game development portfolio to improve my Unity and C# skills by building a complete playable horror game with gameplay systems, UI, audio, and enemy AI.
